Flex-Flow

A propriedadae flex-flow é uma abreviação das propriedades (Flex-Direction e Flex-wrap) do flexbox. Ela permite definir as duas propriedades em um único valor. Exemplo:

Flex-flow: < valor de: flex-direction > + < valor de: flex-wrap >

O valor padrão de flex-flow são os valores padrões do flex-direction + flex-wrap como no exemplo a seguir:

Flex-flow: now nowrap;
