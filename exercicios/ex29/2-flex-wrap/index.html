Flex-Wrap

É uma propriedade de Flex-Box, que define como os elementos filhos de um container flexivel devem ser dispostos em relação a linha principal (Main Axis) Quando não há mais espaço suficiente para acomodala em uma única linha

Nowrap (Valor padrão)

Os elementos filhos são dispostos em uma única linha que mesmo que isso significa que o container flexivel fique maior do que o espaço disponivel
