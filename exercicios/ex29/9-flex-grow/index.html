Flex-Grow

Essa propriedade é usada em container flex para controlar a distribuição de espaço disponivel entre os elementos filhos flexiveis. Quando aplcada a um elemento filho flexivel a propriedade flex-grow especifica aproporção em que um elemento deve crescer em relação aos outros elementos flexiveis dentro do container. Por exemplo, se dois elementos filhos flexiveis tiverem os valores diferentes para flex-grow o elemento com o valor mais alto crecerá mais em relação ao elemento com valor mais baixo
