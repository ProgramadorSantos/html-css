Flex-Shrink

O valor pradrão da propriedade Flex-Shrink é 1, o que significa que os elementos filhos flexiveis reduzirão de tamaho proporcionalmente quando houver falta de espaço disponivel no container pai. No entanto é posivel definir o valor como 0, o que impede que o elemento filho reduza de tamanho quando não há espaço disponivel. Além disso, é possivel definir valor maior que 1, o que faz com que o elemento filho reduza de tamanho em uma proporção maior do que outros elementos filhos flexiveis no container. A propriedade flex-shrink é normalmente usada em conjunto com outras propriedades flexbox, como (Flex-Grow) e (Flex-Basis), para criar um layout flexivel e responsivo. Juntos essas propriedades permitem que os elementos filhos se ajustem automaticamente ao tamanho do container pai, dependendo do espaço disponivel e das necessidades de layout
