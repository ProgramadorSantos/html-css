Flex-basis

A propriedade é usada em container flex para definir o tamanho inicial de um elemento antes que o espaço disponivel seja distribuidos entre os elementos filhos flexiveis. Quando aplicado a um elemento filho flexivel a propriedade flex-basis especifica o tamanho base o elemento antes que ele seja esticado ou encolhido em relação aos outros elementos filhos flexiveis dentro do container . Por exemplo, se todos os elementos filhos flexiveis tiverem o mesmo valor para flex-basis o espaço disponivel será distribuido igualmente entre eles
