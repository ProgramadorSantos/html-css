Aligm-Self

A propriedade align-self, é uma propriedade que permite ajustar o alinhamento vertical de um elemento filho em relação ao eixo transversal (Y) do container flexivel em que está inserido. Essa propriedade é usada para controlar o alinhamento vertical de um elemento filho individualmente enquanto a propriedade align-items é usado para definir o alinhamento vertical de todos os elementos filhos ao mesmo tempo

Auto

O valor padrão, o elemento filho usa o valor padrão definido pela propriedade "Align-Items"
