Propriedade (Order)

A propriedade order é uma propriedade CSS que permine definir a ordem de exibição dos elementos em um layout flexivel (flex-box)

Normalmente, os elementos em um layout flex-box são exibidos na ordem em que aparecem no codigo HTML. No entanto, as vezes é necessario reorganizar a ordem em que esses elementos são exibidos na página. É aí que a propriedade "ORDER" entra em jogo
