Propriedade Flex-Direction

A propriedade define em qual direção o container deseja empilhar os itens flexiveis. Essa propriedade normalmente pe usada para definir a direção principal (Main Axis) em que os elementos filhos (Chield Elements) de um container flexivel são dispostos

Valor (Row)

Vejamos aqui temos um exemplo do valor padrão (row), que define que os elementos filhos são dispostos em uma linha horizontal da esquerda para a direira

Valor (Column)

O valor column: Empilha os itens flexíveis verticalmente (De cima para baixo) como no Exemplo abaixo:

2
3

O valor (Column-Reverse) faz com que os elementos sejam dispostos em uma coluna vertical de cima para baixo como no Exemplo a abaixo:
